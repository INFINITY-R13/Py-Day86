# Py-Day86: Typing Speed Test

A Tkinter GUI desktop application that tests your typing speed with real-time feedback.

## Features

- **Real-time WPM calculation** - See your words per minute as you type
- **Accuracy tracking** - Monitor your typing accuracy percentage
- **Visual feedback** - Correct characters highlighted in green, incorrect in red
- **Multiple sample texts** - Random selection of practice texts
- **Timer functionality** - Track your typing duration
- **Complete test results** - Final statistics when you finish

## How to Run

```bash
python typing_speed_test.py
```

Or use the launcher:

```bash
python run_typing_test.py
```

## How to Use

1. Click "Start Test" to begin
2. Start typing the displayed text in the input area
3. Watch your WPM and accuracy update in real-time
4. Complete the entire text to see your final results
5. Use "Reset" to try again or "New Text" for a different passage

## Requirements

- Python 3.x
- Tkinter (usually included with Python)

The application features a clean, user-friendly interface with color-coded feedback to help improve your typing skills!
