# Py-Day86: Typing Speed Test

A comprehensive Tkinter GUI desktop application that tests your typing speed with real-time feedback and detailed performance analysis.

## Features

- **Real-time WPM calculation** - See your words per minute as you type
- **Accuracy tracking** - Monitor your typing accuracy percentage with live updates
- **Visual feedback** - Correct characters highlighted in green, incorrect in red
- **5 diverse sample texts** - Random selection from programming, technology, AI, and classic typing texts
- **Timer functionality** - Track your typing duration with precision
- **Enhanced completion results** - Detailed final statistics with performance ratings
- **Performance benchmarking** - Compare your speed against standard typing levels
- **Professional UI** - Clean, modern interface with color-coded feedback

## How to Run

```bash
python typing_speed_test.py
```

Or use the launcher:

```bash
python run_typing_test.py
```

## How to Use

1. **Start**: Click "Start Test" to begin timing
2. **Type**: Start typing the displayed text in the input area below
3. **Monitor**: Watch your WPM, accuracy, and time update in real-time
4. **Complete**: Finish typing the entire passage to see detailed results
5. **Analyze**: Review your performance rating and statistics
6. **Practice**: Use "Reset" to try again or "New Text" for a different passage

## Sample Texts Available

The application includes 5 diverse typing practice texts:
- Classic pangram (contains all alphabet letters)
- Programming and software development
- Technology and continuous learning
- Artificial intelligence and machine learning
- Internet and global communication

## Performance Levels

- **Beginner**: 15-25 WPM
- **Average**: 25-40 WPM  
- **Good**: 40-60 WPM
- **Excellent**: 60+ WPM

## Requirements

- Python 3.x
- Tkinter (usually included with Python)
